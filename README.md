# 📁 File Organizer CLI

Organize seus arquivos automaticamente por tipo/extensão via linha de comando.

## 📦 Estrutura
```
file_organizer_cli/
├── app/
│   ├── main.py
│   └── organizer.py
├── tests/
├── .gitignore
├── requirements.txt
└── README.md
```

## ▶️ Como rodar

```bash
python app/main.py "CAMINHO/DA/PASTA"
```

## ✅ Resultado esperado

Arquivos serão organizados por pastas de acordo com sua extensão.
